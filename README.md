# Principle Component Analysis (PCA) Implementation

This repository contains a Python implementation of **Principal Component Analysis (PCA)**, a powerful dimensionality reduction technique used in machine learning and data science.

##  Project Overview
PCA is used to transform a high-dimensional dataset into a smaller set of uncorrelated variables called **Principal Components**. This project demonstrates the step-by-step process of:
- Data Preprocessing (Standardization)
- Calculating Covariance Matrices
- Computing Eigenvalues and Eigenvectors
- Projecting data onto lower-dimensional subspaces

##  Requirements

- `numpy`
